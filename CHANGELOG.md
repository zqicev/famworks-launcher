# Список изменений

## 1.4.0

### Новое
- **Браузер** - новый раздел в левом меню. Поиск и установка сборок, модов, ресурспаков и шейдеров прямо в лаунчере, из Modrinth и CurseForge. По умолчанию показывает популярное; моды/паки ставятся в выбранную сборку, сборки Modrinth импортируются к вам как своя сборка.
- **Страница обзора** - по клику на элемент в Браузере открывается страница с описанием, галереей, зависимостями, авторами, лицензией и ссылками (у CurseForge полное описание доступно на сайте).
- **Автоустановка зависимостей** - при установке мода из Браузера вместе с ним автоматически ставятся его обязательные зависимости (рекурсивно, в ту же сборку).
- **Вход через Ely.by** - авторизация по аккаунту Ely.by (с поддержкой 2FA) и нативные скины/плащи Ely.by в игре.
- **Импорт сборок Modrinth (.mrpack)** - установка сборки из файла `.mrpack` или прямо из Браузера.
- **Сборки без загрузчика (Vanilla)** - можно создать чистую сборку без модлоадера.

### Улучшения
- **Установка контента через Браузер**: у открытой сборки во вкладках появилась кнопка «Добавить», ведущая в Браузер к нужному разделу (моды/ресурспаки/шейдеры). Отдельные окна «Добавить мод/ресурспак/шейдер» убраны.
- **Выбор куда установить - как в Modrinth**: при установке открывается окно со списком подходящих сборок и выбором версии, вместо выпадающих списков.
- **Единый стиль выпадающих списков** во всём интерфейсе.
- **Создание сборки**: подсказка версий Minecraft при вводе (только существующие), список версий с учётом выбранного загрузчика и понятная ошибка, если под этот загрузчик такой версии нет.
- **Аккаунты**: окно переделано в аккуратное модальное с новым дизайном; правильная подпись типа аккаунта (ELY.BY / оффлайн); кнопки «Назад»; закрытие по клику вне окна.
- **Вкладка «Моды»**: моды сборки видны сразу, ещё до установки (с пометкой «не установлен»).
- Модальные окна больше не закрываются, если случайно вывести курсор из поля наружу (закрытие только по нажатию именно на фон).
- Иконка «развернуть» показывает два квадрата, когда окно уже развёрнуто.

### Исправления
- **Переключение сборок**: устранён случай, когда после переключения на другую сборку показывалась информация о предыдущей (особенно официальной, которая грузится из сети).
- Во время загрузки сборки теперь показывается индикатор загрузки вместо «Выберите сборку».
- **Скачивание Java**: добавлен запасной источник (Azul Zulu) и повторные попытки - Java ставится даже там, где GitHub недоступен (ошибка `ECONNRESET`).
- **Вход Ely.by**: убран таймаут - запрос идёт через сетевой стек Chromium (учитывает системный прокси), с понятным сообщением при проблемах с сетью.
- Мелкие правки выравнивания и поведения интерфейса.
